🚀 WhatsApp-қа қосу нұсқаулығы

1-қадам — Twilio тіркелу

1
twilio.com/try-twilio сайтына кіріп тегін тіркеліңіз
2
Console-ға кіріп Account SID және Auth Token алыңыз
3
Messaging → Try it out → Send a WhatsApp message бетіне өтіңіз
4
Sandbox нөміріне join [ваш-код] жіберіп іске қосыңыз

2-қадам — Railway-ге орналастыру

1
railway.app — GitHub аккаунтыңызбен тіркеліңіз
2
New Project → Deploy from GitHub repo → файлдарды жүктеңіз
3
Variables бөлімінде осы айнымалыларды қосыңыз:
TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, TWILIO_WHATSAPP_NUMBER=+14155238886
4
Deploy болғаннан кейін URL-ді алыңыз (мысалы: https://bot-xxx.railway.app)

3-қадам — Webhook орнату

1
Twilio Console → WhatsApp Sandbox Settings → Webhook URL-ге:
https://сіздің-домен.railway.app/webhook жазыңыз
2
HTTP POST таңдаңыз және сақтаңыз
3
Дайын! Клиент жазса бот 15 секундтан кейін жауап береді 🎉
💡 Маңызды: Sandbox режимде тек сіздің нөмірлеріңіз жазы ала алады. Нақты клиенттерге WhatsApp Business API-ді іске қосу керек (Meta-да тіркелу).
